## Installation

Add this line to your application's Gemfile:

```ruby
gem 'slack_block_kit'
```


And then execute:

    $ bundle


Or install it yourself as:

    $ gem install slack_block_kit

Finally, require this:

```ruby
require 'slack/block_kit'
```
